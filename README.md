# Weather App

A modern, responsive weather application built with React and Vite. This application allows users to search for cities and view real-time weather conditions alongside a polished user interface.

![Weather App](https://img.shields.io/badge/Weather-App-blue)

- **Real-time Weather Data**: Fetches up-to-date weather information.
- **Search Functionality**: Search for any city to look up its weather.
- **Responsive Design**: fully responsive layout ensuring compatibility across devices.
- **Modern UI**: Styled with Tailwind CSS for a sleek and clean look.
- **Video Background**: Dynamic background support.

## üì¶ Installation & Usage

1. **Clone the repository**
   ```bash
   git clone https://github.com/AG6589/weather-app-lication.git
   cd weather-app-lication
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
